# 🪨📄✂️ Pedra, Papel, Tesoura em Python

### 🎯 Objetivo do Projeto
Desenvolvido durante módulo de **Controle de Fluxo** em Python. 
Aplica condicionais `if/elif/else`, loops `while` e biblioteca `random`.

### 🛠️ Habilidades Demonstradas
- **Lógica Condicional:** Regras do jogo implementadas com `if/elif/else`
- **Loops:** `while` para partidas contínuas até o usuário sair
- **Validação de Input:** Trata entradas inválidas do jogador
- **Módulo Random:** Gera jogadas imprevisíveis do computador
- **UX no Terminal:** Mensagens claras de vitória/derrota/empate

### 🚀 Como Executar
```bash
python pedra_papel_tesoura.py
